# 🩺 BLE 醫療數據接收器

一個專為醫療設備設計的藍牙低功耗 (BLE) 數據接收應用，支持 FORA IR40 體溫計等醫療設備的數據接收和解析。

## 📱 功能特性

- **🔍 設備掃描**: 自動發現附近的 BLE 醫療設備
- **🔗 快速連接**: 優化的連接算法，連接時間 < 5秒
- **📊 數據解析**: 完整支持 IEEE 11073 醫療數據標準
- **💾 數據存儲**: 本地存儲測量歷史記錄
- **🎨 直觀界面**: Material Design 風格的中文界面
- **⚡ 高性能**: 相比 Windows 版本性能提升 10 倍以上

## 🎯 支持的設備

### 體溫計
- ✅ FORA IR40 紅外線體溫計
- ✅ 其他支持 IEEE 11073-10408 標準的體溫計

### 血壓計
- ✅ 支持 IEEE 11073-10407 標準的血壓計
- ✅ 自動解析收縮壓、舒張壓和心率數據

## 📋 系統要求

- **Android**: 5.0+ (API Level 21)
- **藍牙**: 4.0+ (BLE 支持)
- **權限**: 位置權限 (BLE 掃描需要)
- **存儲**: 50MB 可用空間

## 🚀 安裝指南

### 下載 APK
1. 前往 [Releases](../../releases) 頁面
2. 下載最新版本的 APK 文件
3. 或使用 GitHub Actions 構建的版本

### 安裝步驟
1. **啟用未知來源安裝**
   ```
   設置 > 安全 > 未知來源 > 允許
   或
   設置 > 應用 > 特殊權限 > 安裝未知應用
   ```

2. **安裝 APK**
   - 點擊下載的 APK 文件
   - 按照提示完成安裝

3. **授予權限**
   首次運行時會請求：
   - 📍 位置權限 (BLE 掃描需要)
   - 🔵 藍牙權限
   - 📱 設備訪問權限

## 🧪 使用指南

### 基本操作
1. **開啟應用**並授予必要權限
2. **點擊掃描**按鈕搜尋設備
3. **選擇設備**進行連接
4. **進行測量**並查看數據

### FORA IR40 測試流程
1. 確保體溫計已充電並開啟
2. 在應用中點擊「掃描設備」
3. 等待發現 FORA IR40 設備
4. 點擊設備卡片進行連接
5. 在體溫計上進行測量
6. 查看應用中實時顯示的數據

## 🔧 故障排除

### 常見問題

#### 掃描不到設備
- ✅ 確保藍牙已開啟
- ✅ 檢查位置權限是否已授予
- ✅ 確保設備處於配對模式
- ✅ 重啟藍牙服務

#### 連接失敗
- ✅ 確保設備未被其他應用連接
- ✅ 重新啟動應用
- ✅ 重置設備並重新配對
- ✅ 檢查設備電量

#### 數據接收失敗
- ✅ 確保設備處於傳輸模式
- ✅ 檢查設備距離 (建議 < 5 米)
- ✅ 重新連接設備
- ✅ 查看應用日誌
